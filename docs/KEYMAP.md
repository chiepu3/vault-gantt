# Vault Gantt キーマップ

## Obsidianコマンド（設定 > ホットキーから変更可能）

| コマンドID | 既定キー | 説明 |
|---|---|---|
| `vault-gantt:open-workbench` | なし | タスク一覧を開く |
| `vault-gantt:create-new-task` | なし | 新しいタスクを作成 |
| `vault-gantt:undo-last-action` | なし | 最後の操作を元に戻す |

## Workbenchビュー内

| キー | 動作 |
|---|---|
| Ctrl+Z | 最後のAPI操作を元に戻す |
| ダブルクリック（セル） | インライン編集を開始 |
| Enter（編集中） | 編集を確定して保存 |
| Escape（編集中） | 編集をキャンセル |
